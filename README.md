# RestaurantListing

## Endpoints
```
GET  /api/restaurants      # Get all restaurants
GET  /api/restaurants/:id  # Get a restaurant
POST /api/restaurants      # Create restaurant
POST /api/restaurants/:id  # Update restaurant
```
